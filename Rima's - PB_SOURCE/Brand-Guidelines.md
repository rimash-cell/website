# TECHNICAL BRAND GUIDELINES — RIMA ALSHOUFI
**V1.0 · Technical Specifications**

These guidelines provide the precise technical standards for the Rima Alshoufi visual identity. Adherence to these rules ensures architectural consistency across all physical and digital touchpoints.

---

## 01 — THE WORDMARK

The wordmark is a typographic signature built on weight contrast.

### Standard Layout
- **First Name (RIMA)**: Hanken Grotesk · Black 900 · Uppercase
- **Last Name (ALSHOUFI)**: Hanken Grotesk · ExtraLight 200 · Uppercase
- **Kerning**: Optical (-25 tracking for RIMA / -10 for ALSHOUFI)

### Clear Space & Scale
- **Clear Space**: 2x the height of the 'R' in RIMA on all sides.
- **Minimum Size (Digital)**: 120px width for the full wordmark.
- **Minimum Size (Print)**: 30mm width for the full wordmark.

### Color Variants
- **Primary**: Near-Black (#0a0a0a) on Warm Off-White (#f7f5f2).
- **Inverted**: Warm White (#faf9f7) on Near-Black (#0a0a0a).
- **Accent**: RIMA in Near-Black · ALSHOUFI in Bronze (#b87333). *Use sparingly for flagship documents only.*

## 03 — TYPOGRAPHY SCALE (HANKEN GROTESK)

### Desktop Scale
- **Display 01**: 80px / 1.0 LH / -4% tracking (900/200 Pair)
- **Heading 01**: 48px / 1.1 LH / -2% tracking (800 weight)
- **Heading 02**: 32px / 1.2 LH / -1% tracking (700 weight)
- **Body Large**: 20px / 1.6 LH / 0% tracking (300 weight)
- **Body Standard**: 16px / 1.6 LH / 0.5% tracking (300 weight)
- **Labels**: 11px / 1.0 LH / +4px tracking (500 weight · Uppercase)

### Mobile Scale
- **Display 01**: 48px / 1.1 LH / -3%
- **Heading 01**: 32px / 1.2 LH / -1%
- **Body Standard**: 16px / 1.5 LH / 0%

## 04 — GRAPHIC ELEMENTS

### Hairlines
- **Weight**: 0.5pt to 1pt (Print) / 1px (Digital).
- **Color**: Light Sand (#c8c2b8).
- **Logic**: Use to separate major sections or define the "grid of silence."

### The Bronze Glow
- **Effect**: Subtle Radial Gradient.
- **Color**: #b87333.
- **Opacity**: 7% to 13% maximum.
- **Usage**: Only behind the Primary Wordmark or behind a single "Hero Project" image.

---

## 05 — USAGE PROHIBITIONS (DON'TS)

- **Do Not** use cold blues, neon colors, or clinical pure whites (#FFFFFF).
- **Do Not** apply drop shadows to the wordmark or body text.
- **Do Not** skew, stretch, or rotate the wordmark.
- **Do Not** use decorative serifs or "handwritten" fonts.
- **Do Not** use wide-angle real estate photography; stick to "The Lens" atmospheric rules.
